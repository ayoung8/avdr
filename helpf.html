AutoComplete

When putting descriptions in the tables, you may see boxes at the bottom of your screen, with values that you had previously keyed in. If you want to eliminate this feature called AutoComplete, please do the following:

On your browser, click on Tools, Internet Options,. Click on the tab labelled Content. You will see a section called Personal Information. Click on AutoComplete, you will see Forms has a check mark beside it, remove it by clicking on it. At this point, you can clear all previous forms by clicking on the Clear Forms button. Click Ok, then click OK again.

New Drug

New Drug can be located under the Drug menu option. All mandatory fields must be filled (identified in bold). There are 5 tabs: Ingredient, Labelled Use, Risk, Note, and History.

Ingredient Tab
This tab is used to identify the ingredient. Enter the ingredient name (or part of the ingredient name). The system will verify if there is any ingredient starting with the name you have typed. If there is only one, the system will select it. Otherwise a list will be presented and you may select the proper ingredient. If the ingredient is not in the list or no ingredient matches your criteria, you may create a new ingredient by filling out the English and French description field and clicking on the Add button.
To add the item click on the Add/Modify button. To edit an item click on the edit button next to the item. To delete an item click on the delete button next to the item.

Labelled Use Tab
This tab is use to identify the various labelled use for the drug.
To add the item click on the Add/Modify button. To edit an item click on the edit button next to the item. To delete an item click on the delete button next to the item.

Risk Tab
This tab is use to identify the various risks associated with the use of this drug.
To add the item click on the Add/Modify button. To edit an item click on the edit button next to the item. To delete an item click on the delete button next to the item.

Note Tab
This tab is use to identify the various notes for the drug.
To add the item click on the Add/Modify button. To edit an item click on the edit button next to the item. To delete an item click on the delete button next to the item.

History Tab
This tab is a listing of all the historical changes done (within th VDD system) on this drug. To view the information click on the Retrieve button.

Click on Add to add the drug. The system will then display the drug in Update mode. The Update and Delete buttons are now available, providing you have the required access rights.

Find Drug

Find Drug can be located under the Drug menu option,. This screen allows you to search for a drug using the following fields as search criteria:

Name
Type in the name, the system will search for all drugs that have a brand name or generic name that starts with the value entered. To search for the value anywhere in the name add the % sign before the value you want to search on.

Drug DIN
Type in the DIN, the system will search for all drugs that have a DIN that starts with the value entered. To search for the value anywhere in the DIN add the % sign before the value you want to search on.

Drug Manufacturer
Type in the name, the system will search for all drugs that have a manufacturer name that starts with the value entered. To search for the value anywhere in the manufacturer name add the % sign before the value you want to search on.

Ingredient
Type in the name, the system will search for all drugs that have an ingredient name that starts with the value entered. To search for the value anywhere in the ingredient name add the % sign before the value you want to search on. Drug Class
Select the desired class from the drop down list.

Advanced Search
A text box allows the user may to key in the other fields along with the field conditions. Any valid Oracle functions can also be used.

You can use each field individually or a combination of the fields. Once you click on the Search all drugs meeting the selection criteria will be listed on the left hand side of the page. A maximum of 250 rows will be returned from a search.

Edit Drug

Once the search is completed, select the drug or it's associated adverse reaction report by clicking on it (in the left panel). Providing you have access rights to update and/or delete, the Update and/or Delete buttons are available. For more information on how to fill out the fields see the New Adeverse Reaction or New Drug section.

New Adverse Reaction

New Adverse Reaction can be located under the Adverse Reaction menu option. All mandatory fields must be filled (identified in bold). There are 11 tabs: Animal, Reaction, Drug, System, Veterinarian, Assessment, Action, Keyword, Note, Document, and History.

Animal Tab
This tab is used to identify the information concerning the animal(s).

Reaction Tab
This tab is use to identify the general information concerning the adverse reaction.

Drug Tab
This tab is use to identify the drug(s) associated with the adverse reaction.
To add the item click on the Add/Modify button. To edit an item click on the edit button next to the item. To delete an item click on the delete button next to the item.

System Tab
This tab is use to identify the animal system(s) affected during the adverse reaction.
To add the item click on the Add/Modify button. To edit an item click on the edit button next to the item. To delete an item click on the delete button next to the item.

Veterinarian Tab
This tab is use to identify the veterinarian(s) that worked on the case.
To add the item click on the Add/Modify button. To edit an item click on the edit button next to the item. To delete an item click on the delete button next to the item.

Assessment Tab
This tab is use to identify the assessment of the adverse reaction. This tab is only available in update mode to the user having the proper rights.

Action Tab
This tab is use to identify the regulatory action(s) taken following this adverse reaction. This tab is only available in update mode to the user having the proper rights.
To add the item click on the Add/Modify button. To edit an item click on the edit button next to the item. To delete an item click on the delete button next to the item.

Keyword Tab
This tab is use to identify the keyword(s) associated with this adverse reaction.
To add the item click on the Add/Modify button. To edit an item click on the edit button next to the item. To delete an item click on the delete button next to the item.

Note Tab
This tab is use to identify the note(s) associated with this adverse reaction
To add the item click on the Add/Modify button. To edit an item click on the edit button next to the item. To delete an item click on the delete button next to the item.

Document Tab
This tab is use to identify to attach document(s) to this adverse reaction. By clicking on the attach document, you may identify the file to be loaded along with a description.
To view an item click on the view button next to the item. To delete an item click on the delete button next to the item.

History Tab
This tab is a listing of all the historical changes done (within th VDD system) on this adverse reaction record. To view the information click on the Retrieve buton.

Click on Add to add the adverse reaction. The system will then display the adverse reaction in Update mode. The Update and Delete buttons are now available, providing you have the required access rights.

Species

Advanced Search
A text box allows the user may to key in the other fields along with the field conditions. Any valid Oracle functions can also be used.

You can use each field individually or a combination of fields. Once you click on Search all adverse reactions meeting the search criteria will be displayed on the left hand side of the screen. A maximum of 250 rows will be return from a search.

Edit Adverse Reaction

Once the search is completed, select the adverse reaction report by clicking on it. Providing you have access rights to update and/or delete, the Update and/or Delete buttons are available. For more information on how to fill out the fields see the New Adeverse Reaction section.

Utilities

My Profile

From the Utilities menu option, you can select the My Profile option. This screen allows each user to tailor the system to their specific needs. The user may change their name (if misspelled), their password and their preferred language. When changing your password, the password must be at least 6 characters and it must start by a letter. To change a User ID or User Role, please see your Administrator.

User Maintenance

Maintenance of the users is only available to the Administrator.

From the Utilities menu option, you can select the Users Maintenance option. The left portion of the screen lists all the available users that can be updated as well as the New User Option.

New User
All the fields are mandatory. Fill in all the fields and click on Add. The User Id can be up to 30 characters and must be unique, the User Name can be up to 30 characters. The system does allow for multiple occurrences of the same User Name. It is the User Id that makes it unique.

Delete A User
From the user list on the left hand side, click on the user and click on the delete button.

Edit a User
From the user list on the left hand side, click on the user that requires updating. All fields can be updated except for the User Id. Click on the Update button to apply any changes.

Table Maintenance

Maintenance of the tables is only available to the Administrator.

From the Utilities menu option, you can select the Table Maintenance option. The left portion of the screen lists all the available tables. The right of the screen displays the current table. To select a table, click on the desired table on the left side of the screen.

Adding a Row
To add a new row to the table, the user must fill out the bottom portion of the screen and click on the Add/Modify button. All mandatory fields will be in Bold format. If the mandatory fields are not entered, the system will prompt you (after clicking on the Add/Modify button) with a message indicating what field has not been entered.

Deleting a Row
Scroll through the table using the scroll bar to locate the row that is to be deleted.
The last column of the table has a column labelled Action. There are 2 available actions under this column, E (Edit) or D (Delete). Click on the D (Delete) to delete the row. The system will prompt with a message asking if you are sure you want to delete the row. Click OK to continue.
Delete is only allowed on rows that are not being used by the application.

Edit a Row
Scroll through the table using the scroll bar to locate the row that is to be edited. The last column of the table has column labelled Action. There are 2 available actions under this column, E (Edit) or D (Delete). Click on the E (Edit) to edit the row. The row will appear at the bottom of the screen. To save any changes, you must click on the Add/Modify button.

Logout

This menu option allows the user to logout from the application.

Francais/English

This menu option allows you to toggle from English to French or vice versa.
